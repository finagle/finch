The `finch-json4s` module provides the Finch library a support for the [JSON4S](http://json4s.org/) library.

Installation
------------
Use the following _sbt_ snippet:

```scala
libraryDependencies ++= Seq(
  "com.github.finagle" %% "finch-json4s" % "0.6.0"
)
```
